When creating a user on Seamless Karma, one has to specify their organization - either by specifying the organizations name or seamless karma id. Allocation in this case is an optional field: if nothing is specified, it'll default to the usual value for that organization.

Additionally, one can create a new organization by just adding their name here with a new Seamless Karma user - in this case, the allocation field is required, and will be the default allocation for new users from that organization.
